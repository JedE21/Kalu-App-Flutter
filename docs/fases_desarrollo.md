\# Fases de Desarrollo - Kalú App Flutter



\## Fase 1: Estructura base

Crear carpetas, tema visual, colores y configuración inicial de main.dart.



\## Fase 2: Splash Screen

Crear pantalla inicial con nombre Kalú, frase y navegación al Home.



\## Fase 3: Home principal

Crear pantalla principal tipo tienda online con buscador, banner, categorías, productos destacados y barra inferior.



\## Fase 4: Datos mock

Crear modelos y datos de prueba para categorías y productos.



\## Fase 5: Categorías

Crear pantalla de categorías y navegación hacia productos por categoría.



\## Fase 6: Productos

Crear listado de productos con tarjetas, precio, imagen temporal y botón de agregar.



\## Fase 7: Detalle de producto

Crear pantalla con imagen, descripción, precio, cantidad y botón para agregar al carrito.



\## Fase 8: Carrito

Crear carrito local con lista de productos, cantidades, subtotal y total.



\## Fase 9: Checkout por WhatsApp

Generar mensaje de pedido para enviar por WhatsApp.



\## Fase 10: Conexión con Supabase

Reemplazar datos mock por datos reales desde Supabase.



\## Fase 11: Pedidos reales

Guardar pedidos en Supabase.



\## Fase 12: Login y perfil

Agregar autenticación y perfil de cliente.



\## Fase 13: Mejoras finales

Animaciones, responsividad, optimización, pruebas y APK.
